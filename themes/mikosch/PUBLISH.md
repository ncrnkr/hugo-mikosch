Mikosch Theme - Quick Publish Blurb

Mikosch is a minimal, accessible Hugo theme ideal for personal blogs and developer portfolios. It features a clean, dark aesthetic, responsive navigation with a mobile toggle, and a ready-to-run `exampleSite` with placeholder content.

Highlights

- Minimal, readable layouts for posts and lists
- Responsive header with mobile menu
- Included exampleSite for quick demos
- MIT licensed

Install

1. Copy the `mikosch` directory into your site's `themes/` folder.
2. Set `theme = "mikosch"` in your `config.toml`.
3. Run `hugo server -D --themesDir themes` to preview.

Note: Replace placeholder images and example content before publishing publicly.
