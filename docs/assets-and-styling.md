# Assets and styling

## CSS and JS

- `themes/mikosch/assets/css/main.css`
- `themes/mikosch/assets/js/main.js`

The theme uses Hugo Pipes to bundle and fingerprint assets automatically.

## Images

Use `static/images/` for your images. Example placeholders live in:

- `static/images/hero-placeholder.svg`
- `static/images/page-*.svg`
- `static/images/post-*.svg`

## Fonts

The theme loads these Google Fonts:

- Fraunces
- Space Grotesk
